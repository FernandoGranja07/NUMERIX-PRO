Fórmula General
Polinomio de Taylor
Pn(x) = f(a) + f'(a)(x−a) + f''(a)/2! (x−a)² + ··· + f(n)(a)/n! (x−a)n
Resto de Lagrange
Rn(x) = f(n+1)(ξ) / (n+1)! · (x−a)n+1, ξ ∈ (a, x)
Error Absoluto
Ea = |f(x) − Pn(x)|
Cota del Resto
|Rn| ≤ M / (n+1)! · |x−a|n+1, M = máx|f(n+1)|

Método de Bisección
Método cerrado · Requiere cambio de signo en [a, b] · Convergencia lineal garantizada
ℹ El sistema divide el rango en subintervalos, detecta cambios de signo por el Teorema de Bolzano y aplica Bisección en cada uno.
Pasos más pequeños detectan más raíces pero tardan más
📐
Método de la Regla Falsa (Falsi)
Método cerrado · Converge más rápido que bisección en la mayoría de los casos
ℹ El sistema escanea el rango, detecta cambios de signo y aplica Regla Falsa en cada subintervalo encontrado.
∂
Método de Newton-Raphson
Método abierto · Derivada numérica automática · Convergencia cuadrática
ℹ La derivada f'(x) se calcula automáticamente por diferencias centrales: f'(x) ≈ [f(x+h) − f(x−h)] / 2h. No se requiere ingresarla manualmente.
ℹ En modo automático, el sistema detecta cambios de signo y usa el punto medio de cada subintervalo como x₀ para Newton-Raphson.
〜
Método de la Secante
Método abierto · No requiere derivada · Orden de convergencia ≈ 1.618 (áureo)
ℹ En modo automático, usa los extremos de cada subintervalo como x₀ y x₁ para el método de la Secante.

Método de Müller

Aproximación cuadrática iterativa. Encuentra raíces reales y complejas. Convergencia automática de orden ≈ 1.84.

📐
Parámetros de Entrada
El sistema evalúa la convergencia automáticamente — no se requiere especificar n
Funciones: sin cos exp ln sqrt — Potencias: ^
Los 3 puntos iniciales deben ser distintos
El método se detiene cuando Ea < tolerancia
📖
Fundamento del Método
Parábola interpolante
Ajusta una parábola por los 3 puntos (x0, f0), (x1, f1), (x2, f2)
Diferencias divididas
b = [x2,x1] + [x2,x0] − [x1,x0]
Nueva raíz estimada
x3 = x2 − 2f2 / (b ± √(b² − 4ac))
Error relativo
Ea = |(xnuevo − x2) / xnuevo| × 100%

Método de Bairstow

Extrae factores cuadráticos (x² − r·x − s) mediante deflación sintética doble. Encuentra todas las raíces reales y complejas conjugadas en una sola pasada.

🔢
Parámetros de Entrada
Ingresa el polinomio como expresión — se extraen los coeficientes automáticamente
Soporta: xⁿ, coeficientes enteros y decimales, signos negativos
El sistema prueba semillas alternativas automáticamente
📖
Fundamento del Método
Factor cuadrático objetivo
Busca (x² − r·x − s) que divide exactamente a P(x)
División sintética doble
bᵢ = aᵢ + r·bᵢ₋₁ + s·bᵢ₋₂ → cᵢ = bᵢ + r·cᵢ₋₁ + s·cᵢ₋₂
Correcciones (Jacobiano)
Δr = (−b_{n−1}·c_{n−2} + b_n·c_{n−3}) / det
Raíces del factor
x = ( r ± √(r² + 4s) ) / 2

Método de Horner

Evalúa P(c) y P′(c) de forma eficiente. La secuencia anidada bₙ = aₙ, bₖ = aₖ + c·bₖ₊₁ da b₀ = P(c). Aplicado al cociente Q(x) da Q(c) = P′(c).

⚡
Parámetros de Entrada
Ingresa el polinomio P(x) y el punto c donde evaluar
Los coeficientes se extraen automáticamente — soporta términos faltantes
Se calculará P(c) y P′(c) = Q(c)
📖
Fundamento del Método
Inicio: bₙ = aₙ
El primer valor es el coeficiente líder
Recurrencia
bₖ = aₖ + c · bₖ₊₁ para k = n−1, …, 0
Resultado: b₀ = P(c)
Cociente Q(x) tiene coefs b₁, b₂, …, bₙ
P′(c) = Q(c)
Segunda aplicación de Horner sobre Q(x) en c
